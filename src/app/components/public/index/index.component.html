About space

Коммуникационная платформа «Starter» представляет собой коммуникационную площадку для менторов, стажеров и владельцев проектов без централизованной системы управления.

Description for Trainees

В роли стажёра вы сможете:

  1. Получить платную консультацию от ментора;
  2. Принять участие в реальном проекте;
  3. Подтвердить свой уровень квалификации;
  4. Получить отзыв и ревью от ментора;
  5. Приумножить опыт в своей области.

Description for Mentors

Description for Customers
